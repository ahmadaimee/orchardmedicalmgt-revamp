How to use the Blog Writer

  1. Fill in the metadata on the left (title, category, date, tags)
  2. Write your article in Markdown in the center editor
  3. Preview updates live on the right as you type
  4. Click Download HTML to export a complete, ready-to-deploy post
  5. Move the downloaded file into the blog/ folder
  6. Add the post to js/blog-data.js so it appears on the blog index
