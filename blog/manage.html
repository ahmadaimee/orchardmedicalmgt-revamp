📚 How to Add a New Post to This Manager

This page reads its data from a JavaScript array called POSTS defined at the top of this file's <script> block. When you publish a new post, add an entry to that array.

  1. Open blog/manage.html in your code editor.
  2. Find the const POSTS = [ array near the top of the <script> tag.
  3. Add a new object to the end of the array following the same structure as the existing entries. Example:

    { title: "Your New Post Title", slug: "your-new-post-slug", category: "Revenue Cycle", categoryKey: "rcm", date: "May 1, 2026", dateIso: "2026-05-01", readTime: "8 min read", status: "published" },
  4. Save the file. Refresh this page in your browser — the post will appear in the table.

Delete Post

This is a static HTML site — there is no server to delete files automatically. Follow these steps to fully remove the post:

Manual Deletion Checklist

  1. Delete the file: Remove from your blog/ folder using your file explorer or code editor.
  2. Remove the card: Open blog/index.html and delete the <article class="blog-card"> block that links to this post.
  3. Remove from sitemap: Open sitemap.xml and delete the <url> entry for this post's URL.
  4. Remove from this manager: Open blog/manage.html, find the POSTS array in the <script> block, and delete the matching entry.
  5. Check related posts: Search for this post's slug in the other blog/*.html files — remove any sidebar or bottom related-post links that reference it.
